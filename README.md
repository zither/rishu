# 协纪·日书

![icon](icon.png)

传统黄历 · 简约呈现

## 介绍

协纪·日书是一款以《协纪辨方书》为依据的传统黄历应用，界面清爽、无广告、无权限。

《协纪辨方书》是清代官方颁布的历法典籍，系统梳理了历代择吉理论，传承至今，被公认为传统黄历的权威依据。

## 特点

- **数据可查**：以《协纪辨方书》为依据
- **界面简约**：清爽设计，无广告干扰
- **隐私保护**：无需任何权限
- **完全免费**：无需付费或内购
